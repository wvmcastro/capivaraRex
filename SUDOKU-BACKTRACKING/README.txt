O codigo fonte se encontra no arquivo "sudoku.c"
#
A compilação pode ser feita através do compilador padrao
da linguagem C, o gcc.
Codigo: gcc sudoku.c -o sudoku
#
Ha dois arquivos de entrada, entrada1.in e entrada2.in
Para rodar o programa basta digitar "./sudoku.c < entrada1.in",
por exemplo, no terminal.

Codigo também disponibilizado em: www.capivararex.wordpress.com
